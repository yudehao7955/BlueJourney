## 一、页面结构

### 1.1 首页（轨迹记录页）
- **入口**：底部导航栏“首页”选项卡，小程序默认打开页
- **核心功能**：
  - 单人/组队模式下的轨迹记录（开始/暂停/停止、实时数据展示、地图轨迹绘制）
  - 组队模式下显示队伍状态及队长控制按钮（出发/结束）
  - 断点续录、智能滤波、停留检测、增量同步
  - 调试面板（可选，由全局设置控制）

### 1.2 组队页
组队页分为两个 **Tab**：
- **Tab1：我的队伍**
  - 展示当前用户作为队长或队员参与的、未结束的队伍（状态为“准备中”或“行进中”）
  - 每个队伍卡片显示：队伍名称、人数、状态（准备中/行进中）、队长/队员标识
  - 点击卡片进入**队伍详情页**
  - 底部提供“创建队伍”按钮

- **Tab2：附近的队伍**
  - 展示公开的、未结束且用户未加入的队伍（状态为“准备中”或“行进中”）
  - 每个队伍卡片显示：队伍名称、队长昵称、人数、状态、距离（km，基于用户当前位置估算）
  - 点击卡片可预览队伍信息并选择“加入”
  - 加入后，该队伍会从“附近的队伍”列表中移除，移至“我的队伍”

### 1.3 队伍详情页
- 从“我的队伍”点击进入
- 功能：队员列表、实时位置地图、轨迹同步、快捷消息、队长操作（出发/结束/踢人）
- 该页面为独立页面，通过导航栏返回按钮退回

## 二、首页用户故事

### 2.1 单人划行用户

| ID | 故事 | 对应功能点 |
|------|------|------------|
| **HOME-1** | 打开首页，地图显示我的当前位置 | 首页加载时请求定位权限，地图定位到当前位置，显示蓝色圆点 |
| **HOME-2** | 点击开始记录，开始采集GPS绘制轨迹 | 点击“开始记录”按钮 → 检查权限 → 创建活动 → 开始GPS采集 → 地图实时绘制轨迹，数据卡片开始刷新 |
| **HOME-3** | 行进中看实时数据 | 数据卡片实时更新总距离、当前速度、时长；地图自动跟随；静止超过2分钟自动暂停记录（停留检测）；卡尔曼滤波平滑轨迹 |
| **HOME-4** | 休息时暂停，继续后恢复 | 点击“暂停”按钮 → 停止记录新点，时长暂停累加；点击“继续”恢复 |
| **HOME-5** | 上岸后停止保存 | 点击“停止”按钮 → 停止监听 → 保存活动到云端，调用高德优化轨迹 → 跳转活动详情页 |

微信小程序运动轨迹采集绘制优化建议报告

1. 引言

在微信小程序中实现运动轨迹记录功能时，开发者常遇到地图绘制卡顿、轨迹与当前位置割裂感严重的问题。本报告基于对佳明（Garmin）和两步路（2bulu）等成熟产品算法的分析，提出一套完整的前端优化方案，并提供可直接运行的微信小程序代码示例，帮助开发者实现流畅、专业的轨迹记录体验。

---

2. 问题分析

2.1 轨迹卡顿的根本原因

· 数据频率与渲染频率不匹配：GPS 定位回调频率可达 1Hz 甚至更高，而微信小程序地图组件的视图更新能力有限，若每次定位都触发 setData 更新 polyline，将导致视图层阻塞。
· 全量数据更新：许多开发者使用 setData({ polyline: [...] }) 全量替换数组，这会触发地图组件重建整个轨迹，性能开销巨大。
· 冗余点过多：未经抽稀的轨迹点随运动时长线性增长，长时间记录后地图组件渲染压力剧增。
· 未过滤异常点：GPS 漂移导致的"飞线"不仅影响美观，还会增加无效数据。

2.2 成熟产品的解决方案

· 佳明（Garmin）：利用硬件加速度计融合 GPS 数据，实现平滑插值；采用双层数据流（原始数据与展示数据分离）。
· 两步路（2bulu）：采用道格拉斯-普克抽稀算法控制渲染点数量；通过时间/距离双阈值过滤；使用定时器低频刷新地图。

---

3. 优化策略

结合以上分析，我们制定以下三条核心优化策略：

3.1 高频采集 + 低频渲染

· 逻辑层：通过 wx.onLocationChange 持续接收所有定位点，将原始点存入内存数组（不触发视图更新）。
· 视图层：启动定时器（如 1 秒/次），将内存中的原始点经过抽稀后，通过差分更新方式（setData 仅修改 polyline[0].points）刷新地图。

3.2 数据分层与过滤

· 原始数据层：存储所有通过阈值过滤的定位点（用于精确距离/时间计算）。
· 渲染数据层：存储抽稀后的点集，仅用于地图绘制。
· 双阈值过滤：设置时间阈值（如 3 秒）和距离阈值（如 10 米），只有满足任一条件才将点存入原始层。
· 异常点剔除：计算相邻点之间的瞬时速度，丢弃超出合理范围（如 50 米/秒）的漂移点。

3.3 抽稀算法控制渲染点数量

· 当原始点数量超过预设阈值（如 1500 点）时，对渲染点集执行道格拉斯-普克抽稀算法，保留路径的关键拐点，剔除直线段冗余点。
· 抽稀后的点集大小控制在几百点以内，大幅降低地图组件渲染压力，同时保证视觉无损。

4. 核心实现

4.1 定位监听与过滤

```javascript
handleLocationChange(location) {
  const now = Date.now();
  const last = this.lastLocation;

  // 异常速度过滤
  if (last) {
    const deltaTime = (now - last.timestamp) / 1000;
    const distance = this.calculateDistance(last, location);
    const speed = distance / deltaTime;
    if (speed > this.config.ABNORMAL_SPEED) return;
  }

  // 时间和距离双阈值
  const timeDiff = last ? now - last.timestamp : Infinity;
  const distanceDiff = last ? this.calculateDistance(last, location) : Infinity;
  if (!last || timeDiff >= this.config.TIME_THRESHOLD || distanceDiff >= this.config.DISTANCE_THRESHOLD) {
    this.rawLocations.push({ ...location, timestamp: now });
    this.lastLocation = { ...location, timestamp: now };
    // 更新总距离等...
  }
}
```

4.2 抽稀算法（道格拉斯-普克）

```javascript
simplifyPoints(points, tolerance = 0.0001) {
  if (points.length <= 2) return points;
  let maxDist = 0, maxIdx = 0;
  for (let i = 1; i < points.length - 1; i++) {
    const dist = this.perpendicularDistance(points[i], points[0], points[points.length - 1]);
    if (dist > maxDist) {
      maxDist = dist;
      maxIdx = i;
    }
  }
  if (maxDist > tolerance) {
    const left = this.simplifyPoints(points.slice(0, maxIdx + 1), tolerance);
    const right = this.simplifyPoints(points.slice(maxIdx), tolerance);
    return left.slice(0, -1).concat(right);
  } else {
    return [points[0], points[points.length - 1]];
  }
}
```

4.3 定时刷新与差分更新

```javascript
refreshMap() {
  if (!this.data.recording) return;
  const renderPoints = this.getRenderPoints(); // 内部调用抽稀
  // 差分更新：只修改 polyline 的 points 属性
  this.setData({
    'polyline[0].points': renderPoints,
    renderPoints: renderPoints.length
  });
}

5.3 核心逻辑（index.js）

```javascript
Page({
  data: {
    currentLat: 39.9042,
    currentLng: 116.4074,
    scale: 16,
    polyline: [{
      points: [],
      color: '#FF5252',
      width: 6,
      borderColor: '#FFFFFF',
      borderWidth: 1
    }],
    recording: false,
    distance: 0,
    duration: 0,
    renderPoints: 0,
    rawPoints: 0,
    hasData: false
  },

  // 内存数据（不触发渲染）
  rawLocations: [],      // 原始点
  lastLocation: null,
  totalDistance: 0,
  startTime: 0,
  timer: null,           // 地图刷新定时器
  timerInterval: null,   // 计时器

  // 配置参数（可调优）
  config: {
    TIME_THRESHOLD: 3000,    // 3秒
    DISTANCE_THRESHOLD: 10,  // 10米
    MAX_RENDER_POINTS: 1500, // 最大渲染点数，超过则抽稀
    ABNORMAL_SPEED: 50       // 异常速度阈值（米/秒）
  },

  onLoad() {
    this.mapCtx = wx.createMapContext('map');
    this.getCurrentLocation();
  },

  onUnload() {
    this.stopRecording();
  },

  // 获取当前位置并移动到地图中心
  getCurrentLocation() {
    wx.getLocation({
      type: 'gcj02',
      success: res => {
        this.setData({
          currentLat: res.latitude,
          currentLng: res.longitude
        });
        this.mapCtx.moveToLocation();
      }
    });
  },

  // ========== 记录控制 ==========
  startRecording() {
    wx.startLocationUpdate({
      success: () => {
        this.setData({ recording: true });
        this.startTime = Date.now();
        wx.onLocationChange(this.handleLocationChange.bind(this));
        this.timer = setInterval(() => this.refreshMap(), 1000);
        this.timerInterval = setInterval(() => this.updateTimer(), 1000);
      },
      fail: err => {
        wx.showToast({ title: '定位失败，请检查权限', icon: 'none' });
        console.error(err);
      }
    });
  },

  stopRecording() {
    if (this.timer) clearInterval(this.timer);
    if (this.timerInterval) clearInterval(this.timerInterval);
    wx.stopLocationUpdate();
    this.setData({ recording: false });
  },

  toggleRecord() {
    this.data.recording ? this.stopRecording() : this.startRecording();
  },

  // ========== 数据处理 ==========
  handleLocationChange(location) {
    const now = Date.now();
    const last = this.lastLocation;

    // 异常点过滤（速度突变）
    if (last) {
      const deltaTime = (now - last.timestamp) / 1000;
      const distance = this.calcDistance(last, location);
      const speed = distance / deltaTime;
      if (speed > this.config.ABNORMAL_SPEED && deltaTime > 0) {
        console.warn('丢弃异常点', speed);
        return;
      }
    }

    // 双阈值过滤
    const timeDiff = last ? now - last.timestamp : Infinity;
    const distanceDiff = last ? this.calcDistance(last, location) : Infinity;
    if (!last || timeDiff >= this.config.TIME_THRESHOLD || distanceDiff >= this.config.DISTANCE_THRESHOLD) {
      const point = {
        latitude: location.latitude,
        longitude: location.longitude,
        timestamp: now
      };
      this.rawLocations.push(point);
      this.setData({ rawPoints: this.rawLocations.length });

      // 更新总距离
      if (last) {
        this.totalDistance += distanceDiff;
        this.setData({ distance: Math.round(this.totalDistance) });
      }
      this.lastLocation = point;

      // 移动地图到当前位置
      this.setData({
        currentLat: location.latitude,
        currentLng: location.longitude
      });
      this.mapCtx.moveToLocation();
    }
  },

  // 计算两点间距离（米）
  calcDistance(p1, p2) {
    const R = 6371000;
    const lat1 = p1.latitude * Math.PI / 180;
    const lat2 = p2.latitude * Math.PI / 180;
    const deltaLat = (p2.latitude - p1.latitude) * Math.PI / 180;
    const deltaLng = (p2.longitude - p1.longitude) * Math.PI / 180;
    const a = Math.sin(deltaLat/2) ** 2 + Math.cos(lat1) * Math.cos(lat2) * Math.sin(deltaLng/2) ** 2;
    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
    return R * c;
  },

  // 定时刷新地图（抽稀 + 差分更新）
  refreshMap() {
    if (!this.data.recording) return;
    const renderPoints = this.getRenderPoints();
    this.setData({
      'polyline[0].points': renderPoints,
      renderPoints: renderPoints.length
    });
  },

  // 获取渲染点集（必要时抽稀）
  getRenderPoints() {
    let points = this.rawLocations.map(p => ({
      latitude: p.latitude,
      longitude: p.longitude
    }));
    if (points.length > this.config.MAX_RENDER_POINTS) {
      points = this.simplifyPoints(points, 0.00005);
      console.log(`抽稀: ${this.rawLocations.length} -> ${points.length}`);
    }
    return points;
  },

  // 道格拉斯-普克抽稀算法
  simplifyPoints(points, tolerance) {
    if (points.length <= 2) return points;
    let maxDist = 0, maxIdx = 0;
    for (let i = 1; i < points.length - 1; i++) {
      const dist = this.perpendicularDistance(points[i], points[0], points[points.length-1]);
      if (dist > maxDist) {
        maxDist = dist;
        maxIdx = i;
      }
    }
    if (maxDist > tolerance) {
      const left = this.simplifyPoints(points.slice(0, maxIdx+1), tolerance);
      const right = this.simplifyPoints(points.slice(maxIdx), tolerance);
      return left.slice(0, -1).concat(right);
    } else {
      return [points[0], points[points.length-1]];
    }
  },

  perpendicularDistance(point, start, end) {
    const dx = end.longitude - start.longitude;
    const dy = end.latitude - start.latitude;
    if (dx === 0 && dy === 0) {
      return Math.hypot(point.longitude - start.longitude, point.latitude - start.latitude);
    }
    const t = ((point.longitude - start.longitude) * dx + (point.latitude - start.latitude) * dy) / (dx*dx + dy*dy);
    if (t < 0) return Math.hypot(point.longitude - start.longitude, point.latitude - start.latitude);
    if (t > 1) return Math.hypot(point.longitude - end.longitude, point.latitude - end.latitude);
    const projX = start.longitude + t * dx;
    const projY = start.latitude + t * dy;
    return Math.hypot(point.longitude - projX, point.latitude - projY);
  },

  updateTimer() {
    if (this.data.recording && this.startTime) {
      const elapsed = Math.floor((Date.now() - this.startTime) / 1000);
      this.setData({ duration: elapsed });
    }
  },

  // ========== 数据持久化 ==========
  saveTrack() {
    if (this.rawLocations.length === 0) {
      wx.showToast({ title: '无轨迹数据', icon: 'none' });
      return;
    }
    const track = {
      id: Date.now(),
      points: this.rawLocations,
      distance: this.totalDistance,
      duration: this.data.duration,
      createTime: new Date().toISOString()
    };
    const tracks = wx.getStorageSync('tracks') || [];
    tracks.push(track);
    wx.setStorageSync('tracks', tracks);
    wx.showToast({ title: '保存成功', icon: 'success' });
    this.setData({ hasData: true });
  },

  clearTrack() {
